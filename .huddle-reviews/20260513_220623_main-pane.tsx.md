## Codex レビュー

- [優先度: 今すぐ] `pointer-events-none` だけでは、サイドバー表示中もオフスクリーンの `main` 内のボタンや入力にキーボードフォーカスできます。`sidebarOpen` かつモバイル時は `main` 配下を `inert`/`aria-hidden` 相当にするか、フォーカスをサイドバー内に閉じ込める実装を追加してください。

- [優先度: 今すぐ] `sidebarOpen` が desktop のサイドバー状態も表す場合、`lg:pointer-events-auto` でクリックは戻りますが、将来 `aria-hidden`/`inert` を単純追加すると desktop のメイン領域まで無効化します。モバイル用の状態名に分ける、または `isMobile && sidebarOpen` の派生値で制御してください。
