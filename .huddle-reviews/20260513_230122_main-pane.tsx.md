## Codex レビュー

- [優先度: 今すぐ] `requestAnimationFrame` がバックグラウンドタブ等で遅延すると、`setTimeout` が先に `false` に戻した後で RAF が `true` を実行し、`playingDetailEnter` が戻らない可能性があります。`timer` は RAF コールバック内で開始するか、`onAnimationEnd` で `setPlayingDetailEnter(false)` する形にしてください。

- [優先度: 後で] `260` が CSS の `animate-mobile-detail-enter` の実時間と二重管理になっています。アニメーション時間を変更すると React 側だけズレるため、`animationend` イベントで解除するか、CSS/TS で共有できる定数に寄せるのが安全です。

- [優先度: 後で] クラス名は `mobile-detail-enter` ですが、`lg:` 条件では無効化されていません。デスクトップでも `detailEnterVersion` が増える経路があるなら意図せずアニメーションします。`motion-safe:max-lg:animate-mobile-detail-enter` のようにモバイル幅かつ reduced motion 非対象に限定してください。
