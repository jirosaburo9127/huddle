## Codex レビュー

- [優先度: 今すぐ] `showAlbums` の表示条件が `channel.is_dm` と連動していません。メニュー項目は DM では非表示ですが、アルバム表示中に DM チャンネルへ遷移すると、状態が残って DM の `channelId` で `ChannelAlbums` が描画され得ます。`{showAlbums && !channel.is_dm && (...)}` にするか、`useEffect(() => setShowAlbums(false), [channel.id])` で閉じてください。

- [優先度: 後で] `ChannelAlbums` が画像一覧・アップロード・モーダル等で重いコンポーネントなら、`channel-view.tsx` の初期バンドルに常時含まれます。`next/dynamic` で `showAlbums` 時だけ読み込む形にすると、通常のチャンネル表示への影響を抑えられます。

- [優先度: 後で] テストが不足しています。最低限、非 DM のオーバーフローメニューで「アルバム」を押すと `ChannelAlbums` が開くこと、DM ではアルバムが表示されないこと、チャンネル切替時に古いアルバムモーダルが残らないことを追加してください。
